linux服务器出现报错：cannot create temp file for here-document:read-only file system

2024-11-30 1 min read # linux

由于一次停电，导致了树莓派非正常关机，重启之后，宝塔面板什么的都打不开了，进度root权限的时候提示：

cannot create temp file for here-document:read-only file system

经过网上各种查询，最后得出可能是非正常关机导致系统文件错误了，磁盘扇区损坏之类的，解决：

执行命令ext4修复系统文件：

fsck.ext4 -y /dev/sda3

因为sda3是主要的分区，所以尝试修复了sda3

输入命令后，点了yes，然后终端一直在输出y，啥也操作不了。

等了一个晚上之后，还是这样，实在等不了了，就结束了进程，然后reboot，神奇的竟然好了。。
