python进入shell环境

2024-11-30 1 min read

shell是Django提供的互动解释器，你可以在这个指令模式中试验代码是否能够正确执行，是相当方便的工具。 在虚拟环境中输入python manage.py shell就可以进入shell： python (env) E:\django_project\my_blog>python manage.py shell Python 3.7.0 (v3.7.0:1bf9cc5093, Jun 27 2018, 04:59:51) [MSC v.1914 64 bit (AMD64)] on win32 Type "help", "copyright", "credits" or "license" for more information. (InteractiveConsole) >>> 看到>>>表示成功进入shell。 举例：输入下面两行指令就可以轻松删除User数据库： python >>> from django.contrib.auth.models import User >>> User.objects.all().delete()
