.gitignore的内容添加

2024-12-05 2 min read # 程序相关 # git

-- 忽略单个文件
filename.txt

-- 忽略特定扩展名的文件
*.log
*.tmp

-- 忽略特定目录
node_modules/
dist/
pycache/

-- 忽略特定路径的文件
logs/debug.log
build/output/

-- 不忽略特定文件（使用 ! 符号）
!important.log

常见示例

-- Python
pycache/
*.py[cod]
*$py.class
.env
venv/
.venv/
*.pyc

-- Node.js
node_modules/
npm-debug.log
yarn-debug.log
yarn-error.log

-- IDE
.idea/
.vscode/
*.swp
*.swo

-- 编译输出
dist/
build/
*.exe
*.dll
*.so
*.dylib

-- 日志文件
*.log
logs/
log/

-- 系统文件
.DS_Store
Thumbs.db

-- 配置文件
config.ini
settings.json

如果我想忽略bins/run目录下的文件但是不包括该目录下的start.bat和stop.bat应该怎么写

-- 忽略 bins/run 目录下的所有文件
bins/run/*

-- 不忽略 start.bat 和 stop.bat
!bins/run/start.bat
!bins/run/stop.bat

忽略所有目录下的文件

bins/run/*

不忽略特定文件

!bins/run/start.bat
!bins/run/stop.bat

如果有子目录，需要不忽略子目录

!bins/run/*/

然后不忽略子目录中的特定文件

!bins/run//start.bat
!bins/run//stop.bat
