Golang 安装sdl2以及相关依赖

2024-11-30 2 min read # Go

在windows下安装，首先需要安装Mingw，Minimalist GNU on Windows 的缩写。它实际上是将经典的开源 C语言 编译器 GCC 移植到了 Windows 下，并且包含了 Win32API ，因此可以将源代码编译生成 Windows 下的可执行程序。
下载地址：https://www.mingw-w64.org/downloads/
1.找到MingW-W64-builds，点击跳转

2.如果是直接安装版本，则点击Installation: Sourceforge跳转下载.exe文件，但是我这里安装的时候报错了。所以就得下载离线版本

3.点击Sources 下的SourceForge链接
4.进入后，拉到最下面，这里应该可以自己选吧，我下载了x86_64-posix-seh，下载完后直接解压到你自己目录即可

5.配置环境变量，需要设置两个环境变量：H:\Program Files\mingw64\bin; H:\Program Files\mingw64\x86_64-w64-mingw32\bin
6.下载SDL2相关依赖库，SDL2开发库下载地址：http://libsdl.org/download-2.0.php

7.然后把下图目录下的文件全都直接复制到H:\Program Files\mingw64\x86_64-w64-mingw32目录下，如果要下载sdl2_image、sdl2_mixer、sdl2_ttf、sdl2_gfx的话也是同样的方法重复第6和7步

8.下载SDL2的golang三方库，使用命令：go get github.com/veandco/go-sdl2
9.到这里就可以正常使用golang基于SLD2的gui开发了
